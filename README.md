# AI Resume Analyzer

AI Resume Analyzer is a web-based application that analyzes resumes using Natural Language Processing (NLP). The system extracts important technical skills from a resume and calculates a simple resume score based on detected skills.

The application allows users to upload a resume in PDF format. It then processes the document, identifies relevant skills such as Python, Machine Learning, SQL, and other technologies, and displays them along with a calculated resume score.

This project demonstrates how machine learning and NLP techniques can be used to automate resume screening, which is a common task in recruitment systems.

## Features

* Upload resume in PDF format
* Extract skills automatically from resume text
* Display detected skills
* Generate a resume score
* Simple and interactive web interface

## Technologies Used

* Python
* Streamlit
* Natural Language Processing (NLP)
* PDF Text Extraction

## Use Case

This project simulates an automated resume screening tool that could help recruiters quickly analyze resumes and identify candidates with the required skills.

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

python -m streamlit run app.py
